# HealthTrack 前端功能指南

## ✅ 已实现的功能

### 1. 用户注册、登录、登出

**访问方式：**
- **注册**：访问 `http://localhost:3000/register` 或点击登录页的"Sign Up"链接
- **登录**：访问 `http://localhost:3000/login`（未登录时会自动跳转）
- **登出**：
  1. 点击页面右上角的用户头像（圆形图标）
  2. 在下拉菜单中选择 "Logout"

### 2. 记录日常活动

**访问方式：**
- 点击顶部导航栏的 **"ACTIVITIES"** 按钮
- 在Activities页面右下角的 **蓝色圆形 "+" 按钮** 添加新活动

**支持的活动类型：**
- Workout (运动)
- Meal (饮食)
- Steps (步数)
- Sleep (睡眠)
- Hydration (水分)
- Other (其他)

**可记录的数据：**
- 标题和描述
- 计划日期和时间
- 持续时间（分钟）
- 步数
- 卡路里消耗
- 卡路里摄入
- 备注

### 3. 更新活动状态

**方式1 - 单个活动编辑：**
1. 在Activities页面，点击活动卡片右上角的 **三个点菜单** (⋮)
2. 选择 "Edit"
3. 修改状态（Planned, In Progress, Completed, Cancelled）
4. 点击 "Save"

**方式2 - 批量更新：**
1. 在Activities页面，点击活动卡片选中（会有蓝色边框）
2. 选择多个活动后，点击 "Bulk Update Status" 按钮
3. 选择新状态
4. 点击 "Update"

### 4. 删除活动

**删除方式：**
1. 在Activities页面，点击要删除的活动卡片右上角的 **三个点菜单** (⋮)
2. 选择 "Delete"
3. 活动将被立即删除

## 📍 功能位置总结

| 功能 | 位置 | 访问方式 |
|------|------|----------|
| 注册 | `/register` | 未登录时访问，或登录页点击"Sign Up" |
| 登录 | `/login` | 未登录时自动跳转 |
| 登出 | Header菜单 | 右上角头像 → Logout |
| 查看活动 | `/activities` | 顶部导航栏"ACTIVITIES"按钮 |
| 添加活动 | Activities页面 | 右下角蓝色圆形"+"按钮 |
| 编辑活动 | Activities页面 | 活动卡片右上角菜单 → Edit |
| 删除活动 | Activities页面 | 活动卡片右上角菜单 → Delete |
| 更新状态 | Activities页面 | 编辑活动或批量更新 |

## 🎯 快速开始

1. **首次使用**：访问 `http://localhost:3000/register` 创建账户
2. **登录**：使用您的用户名和密码登录
3. **添加活动**：
   - 点击顶部 "ACTIVITIES"
   - 点击右下角 "+" 按钮
   - 填写活动信息
   - 点击 "Save"
4. **管理活动**：
   - 点击活动卡片上的菜单（⋮）进行编辑或删除
   - 或选中多个活动进行批量更新

## ⚠️ 注意事项

- 所有功能需要先登录
- 如果看不到登录/注册页面，请先登出
- 活动创建后可以随时编辑和删除
- 状态更新会立即反映在Dashboard上
